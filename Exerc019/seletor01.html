Exercício 019: Seletor ID

HTML básico

Ao se criar uma 'id' dentro do HTML você deve ficar atento a construção do parâmetro.

Vou dar aqui um parâmetro de como ficaria o uso de um 'id':identidade dentro de um arquivo HTML:
< p id="principal">
< p id="texto01">
< h1 id="primeiro_h1">

Parâmetro Opcional

Este parâmetro usado dentro da folha de estilo é opicional "@charset "utf-8"; "Usado quando você vai usar palavras com caracteres especiais dentro das CSS, algo que dificilmente ocorre.

Outro detalhe importante a se lembrar aqui é que: 'Tudo que em HTMl é id, em CSS vai ser #(cerquilha)

Aprendendo CSS

CSS básico

Obrserve aqui como fica a configuração na folha de STYLE/ESTILO: 

                body{
                background-color: rgb(205,255,203);
                font-family: Arial, Helvetica, sans-serif;
                }
        
                h1{
                    color: rgb(4, 109, 0);
                }
                /* em HTML é id e em CSS vai ser #
                 tudo que é class em HTML vai ser . em CSS*/
                #principal{
                    text-align: center;
                    background-color:rgb(4, 109, 0);
                    color: white;
                }
        
                h2{
                    color: rgb(73, 109, 71);
                }
                p{
                    text-align: justify;
                    text-indent: 30px;
                }

Diferença entre "id" e "class"

O id por mais funcional que pareça, ele é um parâmetro usado somente para um tipo de tag, como por exemplo: < p id="titulo">, e não pode ser usado em outras tags como:

< h2 id="titulo">

Agora, como o parâmetro class, é diferente, poderemos emprega-lo em várias tags diferentes como por exemplo:


< h1 class="principal">
< h2 class="principal">
< h3 class="principal">

E assim por diante...rsss...tome cuidado para não confundir!
