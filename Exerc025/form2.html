Voltar


<label>: serve para associar um texto descritivo a um campo de formulário

Exemplo de Label

        <label for="nome">Nome:</label>

            <input type="text" id="nome" name="nome">

            Aqui, o atributo for do <label> aponta para o id do <input>.

           Assim, se você clicar no texto “Nome:”, o cursor vai direto para o campo de texto.
