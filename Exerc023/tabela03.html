Uso de Tabelas Grandes:

            

                Anatomia para tabelas grandes:
                   1º CAPITION - (DADO)
                   2º TABLE - (TABELA)
                   3º THEAD - (CABEÇA)
                        tr, td e th
                   4º TBODY - (CORPO)
                        tr, td e th
                   5º TFOOT - (RODAPÉ)
                        tr, td e th

O uso de scope - para que serve?

            

                
Em tabelas HTML, as células de 
cabeçalho (th>) são usadas para 
rotular colunas e linhas.
          
            
Em resumo: scope="col" é uma ferramenta 
importante para tornar as tabelas HTML 
mais acessíveis, garantindo que o 
conteúdo seja compreensível mesmo para 
usuários que não podem visualizar a 
tabela de forma visual.

Efeito Zebrado

        

            Este recurso em CSS ajuda a configurar um
            efeito de preenchimento

            tbody > tr:nth-child(2n){
            background-color: lightgray; //pares
            }
            tbody > tr:nth-child(odd){  //impares
            background-color: lightgray;
            }

Título da Tabela Fixo

        

        Elemento sticky - É um elemento que se comporta como 'position relative'. Mas, quando o usuário rola a página e o elemento atinge um limite definido pelo top, left, right ou bottom, ele passa a se comportar como fixed, "grudando" nessa posição.

O que foi acrescentado para ser criar o Efeito?

        

        table{
            position: relative; --adicionado--
            border-collapse: collapse;
            height: 20px;
            width: 300px;
            margin: auto;
        }

        thead > tr > th{      --adicionado--
            position: sticky; --adicionado--
            top: -1px;        --adicionado--
            background-color: rgb(119, 119, 119) ; --adicionado--
        }
