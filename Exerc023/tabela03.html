O uso de scope - para que serve?

                
    Em tabelas HTML, as células de cabeçalho (th>) são usadas para rotular colunas e linhas.

    

    Em resumo: scope="col" é uma ferramenta importante para tornar as tabelas HTML mais acessíveis, garantindo que o conteúdo seja compreensível mesmo para usuários que não podem visualizar a tabela de forma visual.

    background-color: lightgray;

Título da Tabela Fixo

            
    Elemento sticky - É um elemento que se comporta como 'position relative'. Mas, quando o usuário rola a página e o elemento atinge um limite definido pelo top, left, right ou bottom, ele passa a se comportar como fixed, "grudando" nessa posição.

O que foi acrescentado para ser criar o Efeito?

            

    table{
        position: relative; 
        border-collapse: collapse;
        height: 20px;
        width: 300px;
        margin: auto;
    }
    thead > tr > th{      
        position: sticky; 
        top: -1px;        
        background-color: rgb(119, 119, 119); 
    }
