Texto Marcado(MARK)

Podemos criar também um texto marcado usando a tag MARK.

também podemos usar a tag (mark style) para poder marcar um texto com outra cor.



Quando eu tiver várias tags de marcação(MARK)?

Primeiro você deixa todos os textos marcado de amarelo básico

Como eu vou fazer neste exemplo aqui.

Em seguida, na cabeça(HEAD) crie uma tag(STYLE) e entre esta tag crie uma marcação para adicionar o fundo de outra cor.



Exemplo:
"style"
mark {
background-color: linegreen;
}
"/style"

Assim o texto de todas as marcações ficará "verde"
