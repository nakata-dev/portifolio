           Existem Várias maneiras de se configuras o fundo de um Background-image:

        body{
            background-image: url('https://gustavoguanabara.github.io/html-css/imagens/mascote.png');
            background-size: 100px 100px;
            background-repeat: repeat;  //Repete por toda a tela, no valor do tamanho configurado acima
        }
        body{
            background-image: url('https://gustavoguanabara.github.io/html-css/imagens/mascote.png');
            background-size: 100px 100px;
            background-repeat: no-repeat;   //Mostra só um objeto no canto superior esquerdo
        }
        body{
            background-image: url('https://gustavoguanabara.github.io/html-css/imagens/mascote.png');
            background-size: 100px 100px;
            background-repeat: repeat-x;  //Repete na Horizontal
        }
        body{
            background-image: url('https://gustavoguanabara.github.io/html-css/imagens/mascote.png');
            background-size: 100px 100px;
            background-repeat: repeat-y;  //Repete na Vertical
        }
