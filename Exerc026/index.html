Layout Responsivo com Media Queries

Este é um exemplo de como usar **Media Queries** para adaptar o design de um site a diferentes tamanhos de tela. Tente redimensionar a janela do navegador para ver as mudanças acontecerem em tempo real.

A cor de fundo e o estilo deste bloco vão mudar!

Quando a tela for:

  • **Grande:** A cor de fundo será cinza claro.
  • **De Tablet (601px a 900px):** A cor de fundo mudará para um cinza médio e o bloco de informação ficará laranja.
  • **De Celular (até 600px):** A cor de fundo ficará mais escura e o bloco de informação se tornará vermelho.

Você pode usar Media Queries para controlar não apenas cores, mas também layouts, tamanhos de fonte, e até mesmo a visibilidade de elementos.
