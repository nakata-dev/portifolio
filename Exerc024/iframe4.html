Comandos uteis de segurança:

    primeira situação:

    <iframe src="formulario.html" sandbox="sandbox">Seu navegador não é compativel com isso!</iframe> 
    
    segunda situação:
    
    <iframe src="formulario.html" sandbox="sandbox" referrerpolicy="no-referrer"">Seu navegador não é compativel com isso!</iframe>  

    terceira situação:
    
    <iframe src="formulario.html" sandbox="allow-same-origin allow-forms allow-script" referrerpolicy="no-referrer">Seu navegador não é compativel com isso!</iframe>

Usando JavaScript:



Aqui também foram usados os mesmos comando acima citados. O iframe buscar o arquivo iframe5.html e em seguida, configurei os comandos que tornam o script de outros sites mais seguros.



Analise a linha de comando e pratique o uso da mesma, é importante tornar o seu site um pouco mais seguro.
linha de comando:

<iframe src="formulario.html" sandbox="allow-script" referrerpolicy="no-referrer">

allow-script: Serve para permitir script.

referrerpolicy="no-referrer": Serve para não capturar dados de ninguém.
