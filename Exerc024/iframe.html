Como usar o Iframe?

Iframe: É um frame inserido em uma linha, para que você consiga acessar um site dentro deste pequeno retângulo sem sair do site em questão.

Veja aqui a estrutura do iframe para maior compreensão e fixação do conteúdo em si:


<iframe src="http://nakata-dev.github.io/portifolio"frameborder="0">
Mensagem Opicional / link Opicional
</iframe>


O tamanho padrão de um iframe é de: 300x150 pixels

Situação 2: Na parte de edição de uma 'Mensagem Opicional' é possível se colocar um link de acesso.

por Exemplo:

<iframe src="http://nakata-dev.github.io/portifolio"frameborder="0">
<a href="http://nakata-dev.github.io/portifolio"frameborder="0" target="_blank">
</iframe>

Situação 3:

Na CSS você pode configurar o iframe para que ele altere a altura, largura, borda etc.

por exemplo:

    iframe{
        width:400px;
        heitght:auto;
        border: 1px dashed black;
    }
