O uso do Iframe

Abaixo temos um exemplo claro de como inserir um Iframe dentro de um padrão básico de configuração dentro da CSS. Neste caso, você vê um site de link externo.




Esta seria uma situação nova, onde você vê um site de um link local. Onde você busca o caminho do link diretamente dentro da mesma pasta em que criou uma página principal de aprensentação de conteúdos.





Já nesta ultima situação, contemplamos o uso de uma tabela responsiva. Muito útil para aquele cliente que deseja mostrar uma determinada informação de venda sem poupando tempo de seu cliente no ato de ficar abrindo inumeras abas.





Teste com Iframe:

     
Situação 01:

É totalmente possível se inserir um site local para ser mostrado dentro de seu site principal, poupando tempo ao usuário de ficar abrindo novas abas.

Situação 02:

Também é possível você abrir tabelas para poder navegar de forma responsiva dentro de um espaço totalmente limitado por suas configurações da folha de estilos.


Como navegar através de um Iframe?



Para se naver por um iframe, você precisa criar uma lista na qual você pode visualizar uma ou mais páginas dentro de um Iframe já estabelecido.



Neste caso, vamos criar primeir uma lista ao qual vamos poder abrir os iframes acima dentro de um espaço maior de 300px de largura e altura:auto.


Escolha uma opção:


  • Testando o uso do Iframe
  • Página de Teste
  • Tabela Responsiva




Como usar o iframe:


    Anote como fica a estrutura do código acima: 

    <h2>Escolha uma opção:</h2>
    <ul>
        <li><a href="">Testando o uso do Iframe</a></li>
        <li><a href="">Página de Teste</a><li>
        <li><a href="">Tabela Responsiva</a></li>
    </ul>


Configurando o iframe em si:

    
        <iframe id="tela" name="frame" src="">
        <p>Infelizmente sua página não é compatível com isso.</p>
        </iframe>
    
    Atenção: O name deve ser o mesmo que vai no target, caso contrário não vai funcionar o iframe.
