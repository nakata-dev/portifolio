Desafio 07: Imagens Responsivas

O que é uma tag: Picture?

É um container flexível para exibir imagens adaptáveis em diferentes situações. Permite definir várias fontes de imagem e deixar o navegador escolher a mais adequada.


Usado para: Responsividade (diferentes tamanhos para diferentes telas). Formatos diferentes (ex: webp, avif, jpg). Direções artísticas (mostrar outra imagem em telas menores, não só redimensionar).
